## Description
This project implements a scroll-driven hero section animation inspired by a reference demo. It uses GSAP and ScrollTrigger to create smooth, scroll-controlled animations with a focus on performance and responsiveness.

## Features
- Initial load animation for headline and statistics
- Scroll-based movement using GSAP ScrollTrigger
- Responsive layout for different screen sizes
- Optimized animations using transform properties
- Clean and structured React component design

## Tech Stack
- React
- GSAP
- ScrollTrigger
- Tailwind CSS
- Vite
